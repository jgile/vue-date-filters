# Filters for formattins date, datetime, and time.

## Installation

You can install the package via yarn:

```bash
yarn add vue-date-filters
```
or npm:

```bash
npm install vue-date-filters --save
```

## Usage

```js
import VueDates from "vue-date-filters";
Vue.use(VueDates);
```
